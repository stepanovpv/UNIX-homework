# Попробовать вывести с помощью cat содержимое всех файлов в директории /etc. 
# Направить ошибки в отдельный файл в вашей домашней директории. Сколько файлов, 
# которые не удалось посмотреть, оказалось в списке?

pavel@ubuntu:~$ cat /etc/* 2> error.txt

# ... выводится содержимой файлов

pavel@ubuntu:~$ wc -l error.txt
136 error.txt
